Lista de significados

Lista ordenada
A lista ordenada é definida por itens de lista contidos pelas tags "li" e "/li" e estes itens devem estar contidos entre as tags "ol" e "/ol". Cada item da lista deve ter seu conjunto de tags "li - /li", mas a lista toda deve pertencer a um único conjunto "ol - /ol". Para facilitar a memorização, lembre-se: ol - ordered list - lista ordenada.
Lista não ordenada
A lista não ordenada é definida por itens de lista contidos pelas tags "li" e "/li" e estes itens devem estar contidos entre as tags "ul" e "/ul". Cada item da lista deve ter seu conjunto de tags "li - /li", mas a lista toda deve pertencer a um único conjunto "ul - /ul". Para facilitar a memorização, lembre-se: ul - unordered list - lista não ordenada.
Lista de definição
A lista de definição é definida por dois modelos de itens, o termo e sua definição. Por isso, cada item desta lista é um grupo, composto pelas tags "dt - /dt" e "dd - /dd". Depois de completar a lista, estes grupos de elementos devem estar contidos em um único conjunto "dl - /dl". Para facilitar a memorização, lembre-se: dl - definition list - lista de definição. O elemento que funciona como verbete é o dt - definition term - termo da definição e o elemento que funciona como significado é o dd - definition description - descrição da definição.
